# Face Mesh 3D

Веб-приложение, которое захватывает видео с вебкамеры и отображает 3D-маску лица в реальном времени.

## Технологии

- **Three.js** — 3D-рендеринг сцены
- **TensorFlow.js** — запуск модели машинного обучения в браузере
- **MediaPipe Face Mesh** — детекция 478 ключевых точек лица
- **Vite** — сборка и dev-сервер

## Структура проекта

```
src/
├── main.js                  — точка входа, главный цикл
├── FaceTracker.js           — захват камеры и детекция лица
├── Scene3D.js               — 3D-сцена и рендеринг маски
└── facemesh_tesselation.js  — индексы тесселяции маски лица
```

## Запуск

```bash
npm install
npm run dev
```

Открыть в браузере: `http://localhost:5173`

> Требуется разрешение на доступ к камере.

## Управление

| Клавиша | Действие |
|---------|----------|
| `W` | Переключение режима: wireframe / solid |

## Как работает

1. `FaceTracker` запрашивает доступ к камере и загружает модель MediaPipe Face Mesh через TensorFlow.js
2. В каждом кадре модель определяет 478 точек лица
3. `Scene3D` строит 3D-меш из этих точек и рендерит его через Three.js
4. Превью с камеры отображается в правом нижнем углу
